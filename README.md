# cookie-app
Project to understand use of cookies

## Technologies
* Java 17
* IntelliJ IDEA
* Apache Tomcat 10

## Run project
1. Download [Apache Tomcat](https://tomcat.apache.org/download-10.cgi)
2. Run in bin folder (Apache Tomcat) the startup.bat with the command `.\startup.bat`
3. Run the cookie app
